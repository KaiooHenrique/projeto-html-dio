Font

A tag font serve como uma tag de formatação de texto voçe pode mudar a fonte da letra do texto, o tamanho da fonte e por fim a cor do texto, porem a tag foi removida do HTML5 então atualmente voçe pode formatar seu texto usando CSS.

Tag DEL

A tag del e usada para defini uma palavra ou um texto que foi exculido podendo colocar o movito da exclusão data e hora da exclusão

A tag abbr serve para indicar uma abreviatura/siglas no texto, voçe pode adicionar o atributo title para especificar melhor para para o leitor do que se trata a sigla ou palavra abreviada
